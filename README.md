# portfolio-project

This is my portfolio site, but it's also a learning project.

I've built it myself from scratch, using duct tape and Codex. I’m not a trained developer or designer, so a lot of this has been learned by doing. I’ve been using the site to get more comfortable with Codex, HTML, CSS, layout, responsive design, and the basic logic behind JavaScript. 

I’m trying to keep it simple on purpose. Plain HTML, plain CSS, plain JavaScript, and just enough experimentation to keep it interesting.
